## ❌ 需求分析

### 用户需求
"我希望打开菜单能支持多页签显示，即可能同时打开了多个导航菜单，按照多个tab 页显示"

### 需求理解
- 用户希望可以同时打开多个导航菜单
- 使用Tab页签的形式显示已打开的页面
- 可以快速切换不同的页面
- 可以关闭不需要的Tab页
- 类似浏览器的多标签页功能

## ✅ 解决方案

### 1. 状态管理 (Pinia Store)

#### 1.1 扩展Layout Store
在 `layout.js` 中添加多标签页状态管理:

```javascript
export const useLayoutStore = defineStore('layout', {
  state: () => ({
    isCollapsed: false,
    // 多标签页配置
    tabs: [],          // 标签页列表
    activeTab: ''      // 当前激活的标签页
  }),
  // ...
})
```

#### 1.2 标签页操作方法

**添加标签页**:
```javascript
addTab(tab) {
  // 检查是否已存在该标签页
  const existingTab = this.tabs.find(t => t.path === tab.path)
  if (!existingTab) {
    this.tabs.push(tab)
  }
  // 设置为激活状态
  this.activeTab = tab.path
}
```

**移除标签页**:
```javascript
removeTab(path) {
  const index = this.tabs.findIndex(t => t.path === path)
  if (index > -1) {
    this.tabs.splice(index, 1)
    // 如果移除的是当前激活的标签页,激活前一个标签页
    if (this.activeTab === path) {
      if (this.tabs.length > 0) {
        this.activeTab = this.tabs[Math.max(0, index - 1)].path
      } else {
        this.activeTab = ''
      }
    }
  }
}
```

**批量关闭操作**:
```javascript
// 关闭其他标签页
closeOtherTabs(path) {
  this.tabs = this.tabs.filter(t => t.path === path)
  this.activeTab = path
}

// 关闭所有标签页
closeAllTabs() {
  this.tabs = []
  this.activeTab = ''
}

// 关闭左侧标签页
closeLeftTabs(path) {
  const index = this.tabs.findIndex(t => t.path === path)
  if (index > -1) {
    this.tabs = this.tabs.slice(index)
  }
}

// 关闭右侧标签页
closeRightTabs(path) {
  const index = this.tabs.findIndex(t => t.path === path)
  if (index > -1) {
    this.tabs = this.tabs.slice(0, index + 1)
  }
}
```


#### 2.2 核心功能

**点击标签页切换**:
```javascript
const handleTabClick = (path) => {
  layoutStore.setActiveTab(path)
  router.push(path)
}
```

**关闭标签页**:
```javascript
const handleTabClose = (path, event) => {
  event.stopPropagation()
  layoutStore.removeTab(path)
  // 如果关闭的是当前路由,跳转到激活的标签页
  if (route.path === path && activeTab.value) {
    router.push(activeTab.value)
  }
}
```

**下拉菜单操作**:
```javascript
const handleCommand = (command) => {
  const currentPath = activeTab.value
  switch (command) {
    case 'closeOthers':
      layoutStore.closeOtherTabs(currentPath)
      break
    case 'closeAll':
      layoutStore.closeAllTabs()
      router.push('/dashboard')
      break
    case 'closeLeft':
      layoutStore.closeLeftTabs(currentPath)
      break
    case 'closeRight':
      layoutStore.closeRightTabs(currentPath)
      break
  }
}
```

#### 3.2 自动添加标签页
```javascript
// 监听路由变化,自动添加标签页
watch(
  () => route.path,
  (newPath) => {
    // 获取路由元信息
    const meta = route.meta || {}
    const title = meta.title || route.name || '未知页面'
    const icon = meta.icon || null

    // 添加标签页
    layoutStore.addTab({
      path: newPath,
      title: title,
      icon: icon
    })
  },
  { immediate: true }
)
```

## 🎯 功能特性

### 基础功能
- ✅ **自动添加标签页**: 访问新页面时自动创建标签页
- ✅ **点击切换**: 点击标签页快速切换页面
- ✅ **关闭标签页**: 点击关闭按钮关闭标签页
- ✅ **图标显示**: 标签页显示对应图标
- ✅ **激活状态**: 当前页面标签页高亮显示

### 高级功能
- ✅ **关闭其他**: 保留当前标签页,关闭其他所有标签页
- ✅ **关闭所有**: 关闭所有标签页
- ✅ **关闭左侧**: 关闭当前标签页左侧的所有标签页
- ✅ **关闭右侧**: 关闭当前标签页右侧的所有标签页
- ✅ **智能切换**: 关闭当前标签页时自动切换到相邻标签页

### 交互特性
- ✅ **悬停效果**: 标签页悬停时背景色变化
- ✅ **平滑过渡**: 所有操作都有平滑的动画效果
- ✅ **无滚动条**: 标签页区域隐藏滚动条,更美观
- ✅ **响应式设计**: 适配不同屏幕尺寸

## 🚀 可选扩展

### 1. 持久化标签页
使用 localStorage 保存标签页状态:

```javascript
// 保存标签页
watch(
  () => layoutStore.tabs,
  (tabs) => {
    localStorage.setItem('tabs', JSON.stringify(tabs))
  },
  { deep: true }
)

// 恢复标签页
const savedTabs = localStorage.getItem('tabs')
if (savedTabs) {
  layoutStore.tabs = JSON.parse(savedTabs)
}
```

```javascript

  const tab = this.tabs.find(t => t.path === path)
  if (tab) {
    tab.pinned = true
  }
}
```

### 5. 标签页分组
按模块分组标签页:

```javascript
groupTabsByModule() {
  return this.tabs.reduce((groups, tab) => {
    const module = tab.path.split('/')[1]
    if (!groups[module]) {
      groups[module] = []
    }
    groups[module].push(tab)
    return groups
  }, {})
}
```
