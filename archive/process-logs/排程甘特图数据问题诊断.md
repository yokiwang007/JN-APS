## 🔧 可能的问题和解决方案

### 问题1: 字段名不匹配

**症状**: `data.sys_work_centers` 为 `undefined`

**原因**:
- 返回对象中字段名错误
- 可能是 `sys_workCenters` (驼峰) 而不是 `sys_work_centers` (下划线)

**解决方案**:
检查 `mock-extended.js` 中的返回对象:
```javascript
return {
  // 确保使用下划线命名
  sys_work_centers: sysWorkCenters,  // ✅ 正确
  // 而不是
  sys_workCenters: sysWorkCenters,   // ❌ 错误
}
```

### 问题2: 数据生成失败

**症状**: 控制台显示数据生成错误

**原因**:
- `generateSysWorkCenters()` 函数抛出异常
- `generateProductionLines()` 函数返回空数组

**解决方案**:
检查 `mock-extended.js` 中的数据生成函数:
```javascript
export const generateSysWorkCenters = () => {
  const baseLines = generateProductionLines()
  // 检查 baseLines 是否为空
  if (!baseLines || baseLines.length === 0) {
    console.error('❌ generateProductionLines 返回空数组')
    return []
  }
  // ...
}
```

### 问题3: 异步加载问题

**症状**: 页面加载时数据为空,刷新后数据正常

**原因**:
- 数据加载是异步的
- 图表在数据加载完成前就渲染

**解决方案**:
添加 `nextTick` 等待数据加载完成:
```javascript
import { nextTick } from 'vue'

const initializeData = async () => {
  const data = initializeExtendedData()
  productionLines.value = data.sys_work_centers || []
  ganttTasks.value = (data.aps_schedule_tasks || []).map(...)

  await nextTick()
  updateGanttChart()
}
```
