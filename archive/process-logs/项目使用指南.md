# 项目使用指南

## 🚀 快速开始

### 1. 启动项目

```bash
cd aps-frontend
npx vite
```

或者使用npm:

```bash
cd aps-frontend
npm run dev
```

### 2. 访问系统

- 本地访问: http://localhost:5173/
- 网络访问: http://192.168.1.101:5173/

## 🎯 完整业务流程演示

### 流程1: 订单到齐套 (完整流程)

#### 步骤1: 接收订单
1. 访问 `/order/list` 查看订单列表
2. 访问 `/order/split` 进行订单拆解
3. 配置拆解规则 `/order/split-rules`

#### 步骤2: 粗排程
1. 访问 `/coarse-schedule/tasks` 查看待排程任务
2. 执行自动排程或手动排程
3. 访问 `/coarse-schedule/list` 管理排程任务
4. 访问 `/coarse-schedule/gantt` 查看排程甘特图

#### 步骤3: 批次规划
1. 访问 `/batch/merge-rules` 配置揉单规则
2. 访问 `/batch/plan` 执行揉单操作
3. 处理交期红线任务

#### 步骤4: 细排程
1. 访问 `/schedule/batches` 查看待排程批次
2. 访问 `/schedule/optimization` 执行开料优化
3. 查看排版图和优化结果

#### 步骤5: 生产执行
1. 访问 `/production/orders` 下发生产工单
2. 访问 `/production/tracking` 追踪在制品
3. 处理延期预警

#### 步骤6: 齐套管理
1. 访问 `/fulfillment/dashboard` 查看齐套进度
2. 访问 `/fulfillment/slots` 管理齐套货位
3. 处理超期货位
4. 完成发货

---

### 流程2: 按模块演示

#### 综合看板演示
1. 访问 `/dashboard`
2. 查看关键指标(待排程、待揉单、待齐套、交期预警)
3. 查看产能负荷(4条产线)
4. 查看订单状态统计
5. 查看交期预警列表
6. 查看齐套等待列表

#### 订单拆解演示
1. 访问 `/order/split`
2. 点击"同步ERP订单"按钮
3. 在左侧列表中勾选订单
4. 选择拆解规则(默认规则)
5. 点击"执行拆解"按钮
6. 查看右侧生成的生产子订单列表
7. 查看底部BOM零件明细

#### 拆解规则配置演示
1. 访问 `/order/split-rules`
2. 查看现有规则列表
3. 点击"新增规则"按钮
4. 设置规则条件(零件类型、材质、厚度、花色)
5. 设置规则结果(工艺路线)
6. 设置优先级
7. 点击"保存"

#### 粗排程演示
1. 访问 `/coarse-schedule/tasks`
2. 查看左侧待排程子订单列表
3. 查看右侧产线负荷看板
4. 勾选要排程的任务
5. 在底部设置排程参数:
   - 目标产线
   - 计划日期
   - 排程模式(倒排计划/正排计划)
   - 齐套缓冲期
6. 点击"保存排程"按钮
7. 查看产能负荷更新

#### 排程任务管理演示
1. 访问 `/coarse-schedule/list`
2. 使用筛选条件查询任务
3. 查看任务详情
4. 调整任务信息
5. 删除任务

#### 排程甘特图演示
1. 访问 `/coarse-schedule/gantt`
2. 选择日期范围
3. 选择产线
4. 查看甘特图
5. 点击任务查看详情

#### 揉单规则配置演示
1. 访问 `/batch/merge-rules`
2. 配置合并原则(板材利用率优先/设备利用率优先)
3. 配置时间窗规则(提前天数、延后天数、跨越天数)
4. 配置批次规模规则(最小/最大部件数、最大订单数、相似度阈值)
5. 配置交期红线策略(启用/禁用、强制下发预警、最低利用率)
6. 配置智能填缝策略(启用/禁用、利用率阈值、填缝来源)
7. 保存配置

#### 批次规划演示
1. 访问 `/batch/plan`
2. 在左侧列表中勾选待揉单任务
3. 点击"自动揉单"按钮
4. 查看右侧揉单结果
5. 查看批次利用率
6. 处理交期红线任务
7. 查看批次详情

#### 待排程批次演示
1. 访问 `/schedule/batches`
2. 使用筛选条件查询批次
3. 查看批次详情(包含订单明细)
4. 批量排程
5. 查看批次利用率
6. 调整批次

#### 开料优化演示
1. 访问 `/schedule/optimization`
2. 查看优化统计(利用率、板材消耗、优化时间、方案评分)
3. 点击"开始优化"按钮
4. 查看排版图(网格视图/列表视图)
5. 点击"参数设置"配置优化参数
6. 查看排版图详情
7. 打印或下载排版图

#### 生产工单演示
1. 访问 `/production/orders`
2. 查看生产工单列表
3. 查看工单详情(包含工序详情)
4. 下发工单
5. 编辑工单
6. 删除工单

#### 在制品追踪演示
1. 访问 `/production/tracking`
2. 查看在制品追踪列表
3. 使用筛选条件查询
4. 扫码开始工序
5. 扫码完成工序
6. 查看延期预警
7. 处理延期任务
8. 导出数据

#### 齐套看板演示
1. 访问 `/fulfillment/dashboard`
2. 查看齐套统计(待齐套、齐套中、已完成、齐套率)
3. 查看齐套进度表格
4. 切换视图(待齐套/齐套中/已完成)
5. 查看订单详情
6. 开始齐套
7. 完成齐套
8. 查看紧急订单
9. 优先处理紧急订单
10. 导出报表

#### 齐套货位管理演示
1. 访问 `/fulfillment/slots`
2. 查看齐套货位列表
3. 使用筛选条件查询
4. 查看货位详情
5. 释放货位
6. 锁定货位
7. 解锁货位
8. 删除货位
9. 查看超期预警
10. 导出货位

## ⚠️ 注意事项

### 1. 模拟数据
- 所有功能使用模拟数据
- 数据存储在内存中
- 刷新页面后数据重置

### 2. 图表功能
- 完整版Dashboard包含ECharts图表
- 某些浏览器可能需要额外配置
- 简化版Dashboard不包含图表

### 3. 浏览器兼容性
- 推荐使用Chrome、Firefox、Safari
- 不支持IE浏览器

### 4. 网络访问
- 默认监听 localhost:5173
- 可通过 --host 参数配置网络访问
- 防火墙可能需要开放端口

## 🐛 常见问题

### 1. 页面显示空白
- 检查路由配置是否正确
- 检查组件路径是否正确
- 清除浏览器缓存

### 2. 图表不显示
- 检查ECharts是否正确引入
- 检查图表配置是否正确
- 使用简化版Dashboard

### 3. 路由跳转失败
- 检查路由路径是否正确
- 检查路由守卫是否有拦截
- 查看浏览器控制台错误

### 4. 样式不正确
- 检查CSS是否正确引入
- 检查Element Plus是否正确引入
- 清除浏览器缓存
