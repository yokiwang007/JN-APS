# JN-APS 模式切换功能 - 快速指南

## 🎯 当前状态

您的JN-APS系统现在支持两种运行模式!

### ✅ 已完成的功能

1. **模式切换按钮** - 在页面顶部Header区域
2. **API状态检测** - 实时显示后端连接状态
3. **自动模式选择** - 根据配置自动选择API服务
4. **错误提示** - 连接失败时给出详细提示

## 🚀 如何使用

### 查看当前模式

打开浏览器访问 http://localhost:5173/,查看页面顶部:

- 🔵 **蓝色按钮** = 模拟数据模式 (使用localStorage)
- 🟢 **绿色按钮** = 真实API模式 (连接后端)

### 切换模式

1. 点击页面顶部的模式切换按钮
2. 从下拉菜单中选择目标模式
3. 确认切换
4. 页面自动刷新,新模式生效

## 📊 两种模式对比

| 特性 | 模拟数据模式 | 真实API模式 |
|------|------------|-----------|
| 数据存储 | localStorage | MySQL数据库 |
| 后端服务 | 不需要 | 需要 |
| 数据持久化 | 浏览器本地 | 服务器数据库 |
| 适用场景 | 开发、演示 | 生产、测试 |
| 多用户 | 不支持 | 支持 |

## 📝 快速命令

```bash
# 启动前端
cd aps-frontend
npm run dev

# 启动后端
cd aps-backend
npm run dev-v2

# 初始化数据库
cd aps-backend
npm run init-db-v2
```
