## ⚡ 关键代码片段

### 数据格式化（最重要！）
```javascript
// 后端Service
formatXxx(item) {
  const plain = item.get({ plain: true })
  return {
    id: plain.id,
    name: plain.name,
    // 数组字段转换
    items: plain.items ? plain.items.split(',') : [],
    items_str: plain.items  // 同时提供字符串格式
  }
}
```

### 前端数据处理
```javascript
// 处理分页数据
const result = await getXxxItems()
items.value = result.data.list || result.data  // 兼容两种格式
```

### 模式切换
```javascript
// 统一使用api-unified
import { getXxxItems } from '../../utils/api-unified'  // ✅ 正确
import { getXxxItems } from '../../utils/api'         // ❌ 错误
```
