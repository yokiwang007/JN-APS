# 导航菜单分组折叠功能实现报告

## 📊 实现时间
2026-04-05

## ❌ 需求分析

### 用户需求
"导航菜单要可以支持按分组展开和收缩"

### 需求理解
- 用户希望可以控制每个分组(如订单管理、粗排程、批次规划等)的展开/收缩状态
- 点击分组标题时,切换该分组的展开/收缩状态
- 需要平滑的过渡动画
- 需要视觉指示(如箭头图标)显示当前状态

---

## ✅ 解决方案

### 1. 功能设计

#### 1.1 状态管理
使用 Vue 3 的 `reactive` API 管理所有分组的展开/收缩状态:

```javascript
const sectionStates = reactive({
  order: true,              // 订单管理
  coarseSchedule: true,     // 粗排程
  batch: true,              // 批次规划
  schedule: true,           // 细排程
  production: true,         // 生产执行
  fulfillment: true,        // 齐套管理
  monitor: true,            // 监控调整
  strategy: true,           // 策略配置
  capacity: true            // 产能管理
})
```

**设计要点**:
- 所有分组默认展开状态为 `true`
- 使用对象键名标识不同分组
- 响应式数据,状态变化自动触发视图更新

#### 1.2 切换函数
提供切换函数来修改分组状态:

```javascript
const toggleSection = (sectionKey) => {
  sectionStates[sectionKey] = !sectionStates[sectionKey]
}
```

**设计要点**:
- 接收分组键名作为参数
- 使用取反操作切换状态
- 简单高效,一行代码实现

### 2. 模板结构

#### 2.1 分组标题改造
将原有的静态标题改造为可点击的标题栏:

```vue
<!-- 修改前 -->
<div v-show="!isCollapsed" class="nav-section-title">订单管理</div>

<!-- 修改后 -->
<div
  v-show="!isCollapsed"
  class="nav-section-header"
  :class="{ 'nav-section-header--collapsed': !sectionStates.order }"
  @click="toggleSection('order')"
>
  <span class="nav-section-title">订单管理</span>
  <el-icon class="nav-section-icon"><ArrowDown /></el-icon>
</div>
```

**改造要点**:
- 添加 `@click` 事件绑定,点击时触发切换函数
- 添加动态 class,根据状态改变样式
- 添加箭头图标,视觉指示当前状态
- 使用 flex 布局,标题和图标分居两端

#### 2.2 分组内容包裹
将分组菜单项包裹在可折叠的内容区域:

```vue
<!-- 修改前 -->
<RouterLink to="/order/list" class="nav-item nav-item--sub">...</RouterLink>
<RouterLink to="/order/split" class="nav-item nav-item--sub">...</RouterLink>
<!-- ... 更多菜单项 -->

<!-- 修改后 -->
<transition name="section-collapse">
  <div v-show="sectionStates.order" class="nav-section-content">
    <RouterLink to="/order/list" class="nav-item nav-item--sub">...</RouterLink>
    <RouterLink to="/order/split" class="nav-item nav-item--sub">...</RouterLink>
    <!-- ... 更多菜单项 -->
  </div>
</transition>
```

**改造要点**:
- 使用 `<transition>` 组件包裹内容,支持折叠动画
- 使用 `v-show` 控制内容显示/隐藏
- 添加 `nav-section-content` 类,用于样式控制

### 3. 样式实现

#### 3.1 分组标题样式

**样式要点**:
- Flex 布局,标题和图标分居两端
- 添加 `cursor: pointer`,鼠标悬停显示手型
- 添加 `user-select: none`,防止文字被选中
- 添加悬停效果,提升交互体验
- 添加圆角,与菜单项风格一致

#### 3.2 箭头图标样式
```css
.nav-section-icon {
  font-size: 12px;
  color: rgba(255, 255, 255, 0.45);
  transition: transform 0.3s ease;
}

.nav-section-header--collapsed .nav-section-icon {
  transform: rotate(-90deg);
}
```

**样式要点**:
- 图标大小 12px,与标题字体协调
- 半透明颜色,不抢眼
- 添加旋转动画,平滑过渡
- 收缩时旋转 -90 度,向下箭头变为向左箭头

#### 3.3 折叠动画
```css
.nav-section-content {
  overflow: hidden;
}

.section-collapse-enter-active,
.section-collapse-leave-active {
  transition: all 0.3s ease;
  max-height: 500px;
  opacity: 1;
}

.section-collapse-enter-from,
.section-collapse-leave-to {
  max-height: 0;
  opacity: 0;
}
```

**动画要点**:
- 使用 `max-height` 实现高度过渡
- 使用 `opacity` 实现透明度过渡
- 过渡时间 0.3s,平滑流畅
- `max-height: 500px` 是一个足够大的值,确保内容能够完全展开

## 🎨 用户体验

### 使用场景

#### 场景1: 减少视觉干扰
用户在"订单管理"模块工作时,可以收缩其他不相关的分组,减少视觉干扰,专注当前工作。

#### 场景2: 快速导航
用户可以快速展开需要的分组,收缩不需要的分组,提高导航效率。

#### 场景3: 个性化配置
用户可以根据自己的使用习惯,调整分组的展开/收缩状态,实现个性化配置。

### 交互流程

```
1. 用户看到所有分组默认展开
   ↓
2. 用户点击某个分组标题(如"粗排程")
   ↓
3. 该分组标题的箭头图标向下旋转90度
   ↓
4. 该分组的内容区域开始收缩(动画效果)
   ↓
5. 收缩完成后,内容区域隐藏
   ↓
6. 用户再次点击该分组标题
   ↓
7. 箭头图标旋转回原始位置
   ↓
8. 内容区域开始展开(动画效果)
   ↓
9. 展开完成后,内容区域显示
```

## 🚀 可选扩展

### 1. 持久化状态
使用 localStorage 保存用户的折叠状态:

```javascript
// 保存状态
localStorage.setItem('sidebarSectionStates', JSON.stringify(sectionStates))

// 恢复状态
const savedStates = localStorage.getItem('sidebarSectionStates')
if (savedStates) {
  Object.assign(sectionStates, JSON.parse(savedStates))
}
```

### 2. 全部展开/收缩按钮
添加快捷按钮:

```vue
<el-button @click="expandAll">全部展开</el-button>
<el-button @click="collapseAll">全部收缩</el-button>
```

```javascript
const expandAll = () => {
  Object.keys(sectionStates).forEach(key => {
    sectionStates[key] = true
  })
}

const collapseAll = () => {
  Object.keys(sectionStates).forEach(key => {
    sectionStates[key] = false
  })
}
```

### 3. 自动折叠
点击菜单项时自动折叠其他分组:

```javascript
const toggleSection = (sectionKey) => {
  Object.keys(sectionStates).forEach(key => {
    sectionStates[key] = (key === sectionKey) ? !sectionStates[key] : false
  })
}
```

### 4. 手风琴模式
同一时间只能展开一个分组:

```javascript
const toggleSection = (sectionKey) => {
  const isExpanding = !sectionStates[sectionKey]
  Object.keys(sectionStates).forEach(key => {
    sectionStates[key] = (key === sectionKey) ? isExpanding : false
  })
}
```
