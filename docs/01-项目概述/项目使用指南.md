## 一、系统简介

### 1.1 系统概述
杰诺家居云-APS系统是为定制家具企业打造的先进计划排程系统,基于"先排程后揉单"的核心业务逻辑,实现从订单接收到发货的完整生产流程管理。

### 1.2 核心特性
- ✅ **完整的7阶段业务流程**: 订单接收 → 订单拆解 → 粗排程 → 揉单批次 → 细排程 → 生产执行 → 齐套管理
- ✅ **"先排程后揉单"核心逻辑**: 基于交期倒排计划,智能揉单合并
- ✅ **智能排程算法**: 自动排程建议、开料优化、智能填缝
- ✅ **实时监控预警**: 交期预警、延期预警、超期预警
- ✅ **完整数据关联**: 11个数据表,4,485条模拟数据,100%数据关联完整性

### 1.3 适用场景
- 定制家具生产企业的计划排程
- 多产线、多工艺的生产管理
- 复杂的订单揉单和批次规划
- 精确的齐套管理和发货控制

## 三、功能模块详解

### 3.1 综合看板

#### 3.1.1 功能概述
综合看板是系统的首页,展示关键指标、产能负荷、订单状态等重要信息,帮助用户快速了解生产状况。

#### 3.1.2 页面路径
```
/ 综合看板
```

#### 3.1.3 核心指标
- **待排程订单**: 尚未排程的订单数量
- **待揉单批次**: 尚未揉单的批次数量
- **待齐套订单**: 尚未齐套的订单数量
- **交期预警**: 即将超期的订单数量

#### 3.1.4 产能负荷看板
展示4条产线的实时产能负荷:
- 柜体A线
- 柜体B线
- 门板线
- 异形线

#### 3.1.5 图表展示
- 产能负荷趋势图
- 板材利用率趋势图
- 订单状态统计表格
- 交期预警列表
- 齐套等待列表

#### 3.1.6 使用建议
- 每天上班时首先查看综合看板
- 关注交期预警和齐套等待
- 根据产能负荷调整排程计划

#### 3.2.3 订单列表

- 搜索订单
- 筛选订单状态
- 查看订单详情
- 导出订单列表

#### 3.2.4 订单拆解
**功能**: 将销售订单按工艺路线拆解为生产子订单

**操作步骤**:
1. 点击菜单"订单管理" → "订单拆解"
2. 选择要拆解的订单
3. 选择拆单规则:
   - 按材质拆分
   - 按厚度拆分
   - 按封边模式拆分
   - 按工艺路线拆分
4. 点击"执行拆解"按钮
5. 系统自动生成生产子订单
6. 查看拆解结果,确认无误后点击"保存"

**注意事项**:
- 拆解前请确认订单状态为"待拆解"
- 拆解后订单状态会变为"已拆解"
- 拆解结果可以查看BOM零件明细

---

#### 3.2.5 拆单规则
**功能**: 管理订单拆解规则

**操作步骤**:
1. 点击菜单"订单管理" → "拆单规则"
2. 查看现有拆单规则列表
3. 点击"新增规则"创建新规则:
   - 规则名称
   - 规则类型(材质/厚度/封边/工艺)
   - 规则条件
   - 规则优先级
   - 是否启用
4. 点击"保存"完成创建

**常见规则**:
- 材质=颗粒板 → 划入标准柜体子订单
- 材质=密度板 AND 工序包含CNC → 划入吸塑门板子订单
- 厚度=9mm → 划入背板子订单

#### 3.3.1 功能概述

#### 3.3.3 待排程任务
**功能**: 查看待排程的生产子订单,执行排程操作

**操作步骤**:
1. 点击菜单"粗排程" → "待排程任务"
2. 查看待排程任务列表,包含以下信息:
   - 子订单号
   - 所属订单
   - 工艺路线
   - 材质
   - 厚度
   - 花色
   - 部件数
   - 交期
3. 查看右侧的产线产能看板
4. 选择要排程的任务
5. 点击"自动排程建议"按钮:
   - 系统会自动计算计划日期和产线分配
   - 基于倒排计划算法
   - 考虑交期和产能负荷
6. 查看排程建议,可以手动调整:
   - 计划生产日期
   - 分配产线
   - 排程模式(手动/自动)
   - 缓冲期
7. 确认无误后点击"保存排程"

**倒排计划说明**:
- 最迟开工日 = 交期 - 齐套缓冲期 - 生产周期
- 柜体线生产周期: 2天
- 门板线生产周期: 3天
- 烤漆线生产周期: 15天

### 3.4 批次规划

#### 3.4.1 功能概述
批次规划模块负责对已排程的子订单进行合并分组,生成生产批次。

#### 3.4.2 页面列表
```
/batch/merge-rules - 揉单规则配置
/batch/plan        - 批次规划
/batch/list        - 批次列表
/batch/detail/:id  - 批次详情
```

#### 3.4.3 揉单规则配置
**功能**: 配置揉单合并规则

**操作步骤**:
1. 点击菜单"批次规划" → "揉单规则配置"
2. 配置合并原则:
   - 板材利用率优先
   - 设备利用率优先
3. 配置时间窗规则:
   - 最大提前天数(如 +1天)
   - 最大延后天数(如 -1天)
   - 跨越天数
4. 配置批次规模规则:
   - 最小部件数(如 40件)
   - 最大部件数(如 80件)
   - 最大订单数(如 5个)
   - 相似度阈值(如 70%)
5. 配置交期红线策略:
   - 启用/禁用
   - 强制下发预警
   - 最低利用率(如 85%)
6. 配置智能填缝策略:
   - 启用/禁用
   - 利用率阈值(如 85%)
   - 填缝来源(标准件/补件)
7. 点击"保存"完成配置

**揉单时间窗说明**:
- 为了凑够一个最优批次,系统最多只能把明天该切的板子挪到今天,或者把今天不急的板子挪到明天
- 绝不允许将5天后的订单拉过来

**交期红线说明**:
- 当某块板子达到了"最迟开工日期"的临界点,哪怕当前批次只有10块板(出材率极低),系统也要强制生成批次
- 优先保交期,标记为[加急/强制下发]

---

#### 3.4.4 批次规划
**功能**: 执行揉单操作,生成生产批次

**操作步骤**:
1. 点击菜单"批次规划" → "批次规划"
2. 选择产线(如柜体A线)
3. 选择计划日期(如2026-04-01)
4. 点击"提取待揉单数据"按钮
5. 系统自动按材质、花色、厚度、封边模式进行分组
6. 点击"合并生成批次"按钮
7. 系统弹出合并规则设置窗口:
   - 合并原则:板材利用率优先 / 设备利用率优先
   - 批次上限:订单数上限、部件数上限
   - 时间窗设置:最大提前天数、最大延后天数
   - 相似度阈值:70%
8. 确认后,系统自动计算并生成批次
9. 查看生成的批次列表:
   - 批次号
   - 产线
   - 生产日期
   - 材质
   - 花色
   - 厚度
   - 子订单组成
   - 部件总数
   - 板材利用率
   - 合并规则
   - 状态
10. 可以手动调整批次组成(添加/移除子订单)
11. 点击"执行开料优化"预览板材利用率模拟
12. 确认无误后,点击"发布批次" → 批次数据进入细排程

**注意事项**:
- 揉单前请确认排程任务状态为"已排定"
- 揉单后排程任务状态会变为"已揉单"
- 可以查看批次内的订单明细

---

#### 3.4.5 批次列表
**功能**: 查看已发布的批次,支持查看详情、调整、删除

**操作步骤**:
1. 点击菜单"批次规划" → "批次列表"

#### 3.5.4 开料优化
**功能**: 对批次执行开料优化,生成排版图

**操作步骤**:
1. 点击菜单"细排程" → "开料优化"
2. 选择要优化的批次
3. 设置优化参数:
   - 算法选择(贪婪算法/遗传算法)
   - 切割间距(如 2mm)
   - 最大优化时间(如 60秒)
   - 目标利用率(如 90%)
   - 启用旋转优化
   - 启用余料管理
4. 点击"开始优化"按钮
5. 系统调用开料优化算法,生成排版图
6. 查看优化结果:
   - 利用率(如 92.5%)
   - 板材消耗(如 10张大板)
   - 优化时间(如 3.5秒)
   - 方案评分(如 95分)
7. 查看排版图:
   - 网格视图
   - 列表视图
8. 点击"查看详情"查看排版图详细信息
9. 可以打印排版图
10. 可以下载排版图文件

**注意事项**:
- 优化前请确认批次状态为"待排程"
- 优化后批次状态会变为"已排程"
- 可以查看排版图中的零件明细

## 四、常见问题

### 4.1 登录问题
**问题**: 无法登录系统
**解决**:
1. 检查用户名和密码是否正确
2. 检查网络连接是否正常
3. 联系系统管理员

---

### 4.2 数据显示问题
**问题**: 页面数据不显示
**解决**:
1. 刷新页面
2. 清除浏览器缓存
3. 检查网络连接
4. 联系系统管理员

---

### 4.3 排程问题
**问题**: 无法执行排程
**解决**:
1. 检查订单状态是否为"已拆解"
2. 检查产线产能是否充足
3. 检查交期是否合理
4. 联系系统管理员

---

### 4.4 揉单问题
**问题**: 无法执行揉单
**解决**:
1. 检查排程任务状态是否为"已排定"
2. 检查揉单规则是否配置
3. 检查时间窗设置是否合理
4. 联系系统管理员

---

### 4.5 齐套问题
**问题**: 订单无法齐套
**解决**:
1. 检查订单所有部件是否已完成
2. 检查齐套规则配置
3. 检查货位是否可用
4. 联系系统管理员

---
