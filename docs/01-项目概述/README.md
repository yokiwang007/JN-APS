# 杰诺家居云-APS系统

## 📋 项目简介

杰诺家居云-APS(Advanced Planning and Scheduling)系统是为定制家具企业打造的先进计划排程系统,基于"先排程后揉单"的核心业务逻辑,实现从订单接收到发货的完整生产流程管理。

### 核心特性

- ✅ **完整的7阶段业务流程**: 订单接收 → 订单拆解 → 粗排程 → 揉单批次 → 细排程 → 生产执行 → 齐套管理
- ✅ **"先排程后揉单"核心逻辑**: 基于交期倒排计划,智能揉单合并
- ✅ **智能排程算法**: 自动排程建议、开料优化、智能填缝
- ✅ **实时监控预警**: 交期预警、延期预警、超期预警
- ✅ **完整数据关联**: 11个数据表,4,485条模拟数据,100%数据关联完整性

## 🚀 快速开始

- Node.js >= 16.x
- npm >= 8.x

### 安装依赖

```bash
cd aps-frontend
npm install
```

### 启动项目

```bash
npm run dev
```

或使用 Vite 直接启动:

```bash
npx vite
```

## 📊 数据模型

### 数据表统计 (11个表, 4,485条记录)

| 数据表 | 数据量 | 说明 |
|--------|--------|------|
| erp_sales_order | 100 | 销售订单 |
| cad_bom_part | 800 | BOM零件明细 |
| aps_production_order | 300 | 生产子订单 |
| aps_schedule_task | 200 | 排程任务 |
| aps_merge_batch | 50 | 生产批次 |
| aps_batch_detail | 1500 | 批次明细 |
| cam_cutting_pattern | 40 | 排版图 |
| mes_production_order | 35 | 生产工单 |
| mes_wip_tracking | 1000 | 在制品追踪 |
| mes_sorting_slot | 60 | 齐套货位 |
| mes_fulfillment_detail | 400 | 齐套明细 |

### 数据流转链路

```
1. 销售订单 (erp_sales_order)
   ↓ order_id
2. 生产子订单 (aps_production_order)
   ↓ production_order_id
3. 排程任务 (aps_schedule_task)
   ↓ task_id
4. 批次明细 (aps_batch_detail)
   ↓ batch_id
5. 批次 (aps_merge_batch)
   ↓ batch_id
6. 排版图 (cam_cutting_pattern)
   ↓ batch_id
7. 生产工单 (mes_production_order)
   ↓ work_order_id
8. 在制品追踪 (mes_wip_tracking)
   ↓ order_id
9. 齐套货位 (mes_sorting_slot)
   ↓ slot_id
10. 齐套明细 (mes_fulfillment_detail)
```

**数据关联完整性**: 100%
