## 文档使用建议

### 新用户
1. 首先阅读 `/docs/01-项目概述/README.md` 了解项目概况
2. 然后阅读 `/docs/01-项目概述/项目使用指南.md` 学习系统操作

### 产品经理
1. 阅读 `/docs/02-需求设计/优化需求文档.md` 了解业务需求
2. 阅读 `/docs/02-需求设计/主流程与数据流转.md` 了解数据流转

### 开发人员
1. 阅读 `/docs/02-需求设计/优化需求文档.md` 了解系统架构
2. 阅读 `/docs/02-需求设计/主流程与数据流转.md` 了解数据模型
3. 阅读 `/docs/04-数据管理/数据关联验证报告.md` 了解数据关系

### 测试人员
1. 阅读 `/docs/02-需求设计/优化需求文档.md` 了解业务需求
2. 阅读 `/docs/01-项目概述/项目使用指南.md` 了解系统功能
3. 阅读 `/docs/04-数据管理/数据关联验证报告.md` 了解数据验证

### 项目经理
1. 阅读 `/docs/05-项目管理/项目开发完成总结.md` 了解项目成果
2. 阅读 `/docs/01-项目概述/README.md` 了解项目概况
